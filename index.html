Conhecimentos

/* HTML */

é a linguagem padrão de marcação para criar páginas web. Ele fornece a estrutura para o conteúdo na web, utilizando elementos como títulos, parágrafos, imagens, links e mais. Documentos HTML são compostos por elementos aninhados, formando uma estrutura em árvore que os navegadores usam para renderizar o conteúdo.

/* CSS */

é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em HTML. Ela permite a separação entre conteúdo e apresentação, possibilitando que desenvolvedores estilizem páginas web com diversos layouts, cores e fontes. As regras CSS são aplicadas a elementos HTML, definindo como eles devem aparecer na tela do usuário ou em outros meios.

/* JavaScript */

é uma linguagem de programação versátil que adiciona interatividade e comportamento dinâmico às páginas web. Ela permite que os desenvolvedores manipulem o Document Object Model (DOM), respondam a ações do usuário e criem funcionalidades assíncronas. JavaScript é essencial para scripting no lado do cliente, aprimorando a experiência do usuário ao tornar as páginas web mais responsivas.

/* Bootstrap */

é um popular framework front-end que simplifica o processo de design e estilização de páginas web. Desenvolvido pelo Twitter, o Bootstrap fornece um conjunto de componentes pré-projetados, como barras de navegação, grades, botões e formulários, facilitando a criação de uma interface de usuário consistente e responsiva. O sistema de grades e utilitários responsivos do Bootstrap ajuda a garantir que os sites tenham uma boa aparência em vários dispositivos e tamanhos de tela.

/* Git */

um sistema de controle de versão distribuído usado para rastrear alterações no código-fonte durante o desenvolvimento de software. Ele permite que vários desenvolvedores trabalhem em um projeto simultaneamente, mantendo um histórico de alterações e facilitando a colaboração. O Git fornece recursos como branches, merges e rastreamento de versões, ajudando os desenvolvedores a gerenciar e coordenar seu trabalho de forma eficiente.

/* GitHub */

é uma plataforma baseada na web que utiliza o Git para controle de versão e aprimora a colaboração entre desenvolvedores. Desenvolvedores usam o GitHub para hospedar e compartilhar seus repositórios Git, tornando-a uma plataforma popular para projetos de código aberto. O GitHub oferece recursos como pull requests, rastreamento de problemas e wikis, facilitando a comunicação e a gestão de projetos dentro de equipes de desenvolvimento.
